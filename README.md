# Python图书馆管理系统 python788

## 项目概述

本项目是一个基于Python语言的图书馆管理系统，采用前后端分离的设计模式，前端使用Vue.js框架，后端使用Django框架，数据库采用MySQL。

## 技术栈

- 前端：Vue.js
- 后端：Django (Python)
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 首页
- 个人中心
- 用户管理
- 图书馆管理
- 图书信息管理
- 图书类型管理
- 借阅信息管理
- 归还信息管理
- 续借信息管理
- 系统管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Python 3.6+，Django 2.2+
- 数据库：MySQL 5.7+
